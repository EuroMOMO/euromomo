This folder contains additional documents (e.g. design or experiments) for the 
GitHub project. This directory will not be included in the R package.

RStudio setup information:

Either use the https access to the project using:

	https://github.com/thl-mjv/euromomo

This is the preferred way and even works behind firewalls: See.
https://help.github.com/articles/which-remote-url-should-i-use/#cloning-with-https-recommended

Location of the GitHub repository in RStudio which was used for some during the Hackathon:

	git@github.com:thl-mjv/euromomo.git

Or on prompt:
git remote set-url origin git@github.com:thl-mjv/euromomo.git
